  ### Set up the SSH key in Gitlab:
        Step1:
              In server generate sshkeygen:
                 *  ssh-keygen
        Step2:
              Log nto Gitlab:
                Goto account >Settings > SSH Keys
                
        Step3:
              Copy content in server  id_rsa.pub and paste in GitLab.
