# SonarQube:

* What is SonarQube?

      SonarQube is a Software tool which it helps Devlopers to check the quality of the code,
      bugs and Vulnerability(Possibility of the attacks).
  
 ## Features of SonarQube:
      It helps finding the:
      * Code Quality.
      * Bugs Identification.
      * Code Duplication.
      * Code Coverage.
      * Unit Test Cases.
